# TangSim

TangSim is a mobile app for managing small businesses.  
It helps independent merchants manage **products, stock, users**, and **invoices**.  

## Features

- **Products**
  - Add, edit, and delete products
  - Track quantities and categories

- **Stock**
  - View available stock
  - Low stock alerts

- **Users**
  - Manage user accounts
  - Role management (Admin, User)

- **Invoicing**
  - Create invoices
  - Track sales and revenue


## Technologies

- .NET MAUI (C# / XAML)  
- MVVM pattern  
- SQLite database
